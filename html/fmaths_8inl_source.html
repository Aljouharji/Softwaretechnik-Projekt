98 /* In benchmarks conducted circa May 2005 we found that, in the main:
99  * - IA32 machines performed faster with one implementation;
100  * - AMD64 and G4 machines performed faster with another implementation.
101  *
102  * Benchmarks were mainly done with differing versions of gcc.
103  * Results varied with other compilers, optimisation levels, etc.
104  * so this is not optimal, though a tenable compromise.
105  *
106  * Note that the following implementation are NOT what were benchmarked.
107  * We had forgotten to put in overflow detection in those versions.
108  * If you don't need overflow detection then previous versions in the
109  * CVS tree might be worth looking at.
110  *
111  * PS. Don't move the #ifs inside the AL_INLINE; BCC doesn't like it.
112  */

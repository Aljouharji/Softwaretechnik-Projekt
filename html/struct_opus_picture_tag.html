◆ mime_type

char * OpusPictureTag::mime_type

The MIME type of the picture, in printable ASCII characters 0x20-0x7E. The MIME type may also be "-->" to signify that the data part is a URL pointing to the picture instead of the picture data itself. In this case, a terminating NUL is appended to the URL string in data, but data_length is set to the length of the string excluding that terminating NUL.

◆ type

opus_int32 OpusPictureTag::type

The picture type according to the ID3v2 APIC frame:

  1. Other
  2. 32x32 pixels 'file icon' (PNG only)
  3. Other file icon
  4. Cover (front)
  5. Cover (back)
  6. Leaflet page
  7. Media (e.g. label side of CD)
  8. Lead artist/lead performer/soloist
  9. Artist/performer
  10. Conductor
  11. Band/Orchestra
  12. Composer
  13. Lyricist/text writer
  14. Recording Location
  15. During recording
  16. During performance
  17. Movie/video screen capture
  18. A bright colored fish
  19. Illustration
  20. Band/artist logotype
  21. Publisher/Studio logotype

Others are reserved and should not be used. There may only be one each of picture type 1 and 2 in a file.
